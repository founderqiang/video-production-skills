# Timing Presets

## Micro: 2.4-3.2s

Use for short vertical videos or when the hook must start almost immediately.

| Time | Beat |
| --- | --- |
| 0.00-0.20 | Pure black hold |
| 0.20-0.90 | Hero phrase fade/blur reveal |
| 0.90-1.70 | Phrase completes or one word changes |
| 1.70-2.50 | Final phrase hold |
| 2.50-3.20 | Smear, wipe, or cut into body |

## Standard: 4.8-5.8s

Default for tutorials, SaaS/product explainers, and cinematic AI videos.

| Time | Beat |
| --- | --- |
| 0.00-0.30 | Pure black hold |
| 0.30-1.20 | First phrase fades/blur-reveals |
| 1.20-2.20 | First phrase completes and holds |
| 2.20-3.40 | 1-2 word replacements or contrast beats |
| 3.40-4.70 | Final phrase or logo text hold |
| 4.70-5.80 | Transition into next scene |

## Slow: 6.0-7.5s

Use for more cinematic openings or when the next scene needs a strong setup.

| Time | Beat |
| --- | --- |
| 0.00-0.50 | Pure black hold |
| 0.50-1.60 | First phrase slowly appears |
| 1.60-3.00 | Phrase completes and breathes |
| 3.00-4.70 | Two word replacements with horizontal blur |
| 4.70-6.30 | Final phrase or logo lockup hold |
| 6.30-7.50 | White wipe or fade to body |
